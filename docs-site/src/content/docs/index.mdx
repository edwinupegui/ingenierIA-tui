<div class="section-header">
  <h2>Que ofrece ingenierIA TUI</h2>
</div>

<div class="features-grid">
  <div class="feature-card">
    <h3><span class="feature-icon">💬</span> Chat con AI</h3>
    <p>Streaming con GitHub Copilot, Claude API y Mock. 42 slash commands de chat, tool calling, planning mode y context compaction.</p>
  </div>
  <div class="feature-card">
    <h3><span class="feature-icon">⚙️</span> Paleta de comandos</h3>
    <p>26 comandos en <code>:</code> (config, diagnostico, navegacion, exploradores). Separacion estricta: <code>/</code> para chat, <code>:</code> para config.</p>
  </div>
  <div class="feature-card">
    <h3><span class="feature-icon">📄</span> Documentos</h3>
    <p>Navega skills, ADRs, policies, workflows y commands por factory con preview en markdown.</p>
  </div>
  <div class="feature-card">
    <h3><span class="feature-icon">⚡</span> Real-time</h3>
    <p>Eventos SSE en vivo. Health monitoring, tool invocations, enforcement hooks y process monitor.</p>
  </div>
  <div class="feature-card">
    <h3><span class="feature-icon">🤖</span> Multi-Agent</h3>
    <p>SubAgentes aislados, equipos con templates, worktree isolation y mailbox IPC.</p>
  </div>
  <div class="feature-card">
    <h3><span class="feature-icon">🔌</span> IDE Bridge</h3>
    <p>Servidor HTTP local + extension VS Code. Tool approval remoto, sync de contexto y URI handler.</p>
  </div>
  <div class="feature-card">
    <h3><span class="feature-icon">🎨</span> Themes</h3>
    <p>6 variantes (TokyoNight default, Solarized, Gruvbox, Monokai, Matrix, HighContrast). Picker con <code>:theme</code>.</p>
  </div>
  <div class="feature-card">
    <h3><span class="feature-icon">🧩</span> Plugins</h3>
    <p>Sistema extensible con lifecycle hooks. Descubrimiento automatico desde directorio de plugins.</p>
  </div>
  <div class="feature-card">
    <h3><span class="feature-icon">💰</span> Cost Tracking</h3>
    <p>Tokens de entrada/salida/cache en tiempo real. Metricas TTFT/OTPS. Budget alerts al 80% y 100%.</p>
  </div>
</div>

<div class="section-header">
  <h2>Arquitectura Action-Reducer</h2>
  <p>Patron Elm-like con flujo de datos unidireccional</p>
</div>

<div class="arch-diagram">
{`Workers (tokio::spawn)          MPSC Channel           UI (ratatui)
 keyboard.rs ──────┐                     │
 sse.rs ───────────┤                     │
 health.rs ────────┤                     │
 tick.rs ──────────┼── Action ──> ───────┼──> App::handle() ──> AppState ──> render()
 tool_events.rs ───┤                     │       (reducer)       (state)     (pure)
 hook_events.rs ───┤                     │
 cron_worker.rs ───┤                     │    6 crates workspace:
 file_watcher.rs ──┤                     │    domain │ ui │ api │ runtime │ tools
 process_monitor ──┘                     │`}
</div>

## Global

| Tecla | Accion |
|-------|--------|
| `Esc` | Cerrar overlay / volver atras |
| `Ctrl+C` (primero) | Pre-confirmacion: muestra "Presiona Ctrl+C de nuevo para salir" |
| `Ctrl+C` (segundo, dentro de 1.5s) | Sale de la aplicacion |
| `Ctrl+D` | Abrir dashboard |
| `Ctrl+E` | Salir al splash |
| `Ctrl+N` | Inicializar proyecto |
| `/` | Abre autocomplete del chat (solo en chat/splash) |
| `:` | Abre el command palette (cualquier pantalla) |

:::tip[Double Ctrl+C]
ingenierIA TUI usa **double Ctrl+C** para evitar salidas accidentales. El primer `Ctrl+C` arma una confirmacion con ventana de 1.5s; un segundo `Ctrl+C` dentro de ese plazo cierra. Cualquier otra tecla cancela la confirmacion.
:::
